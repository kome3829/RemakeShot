# 概要
## ゲーム概要

プレイヤーを操作して敵を倒しながらステージを進み、
最後に出現するボスを撃破するとゲームクリアとなる
2D縦スクロールシューティングゲーム。

| 画面サイズ | 640×480px |

## ゲームの流れ

本ゲームは、ゲーム開始からリザルトシーンまで一方向に進行するシーン構成となっています。基本的にプレイヤーが任意のシーンへ移動することはできず、ゲームの進行に合わせてシーンが遷移します。
- [シーン遷移図 : SceneTransition.drawio.png](/diaframs/SceneTransition.drawio.png)

#### タイトルシーン

ゲームを起動するとタイトルシーンから開始します。タイトル画像と操作説明画像を確認した後、ゲームシーンへ遷移しゲームを開始します。タイトル画像からゲーム開始への遷移は一方向のみで、タイトル画像、操作説明画像へ戻ることはできません。

- [タイトルシーン : TitleScene_Specification](TitleScene_Specification.md)

#### ゲームシーン

タイトルシーンからゲームを開始すると、ゲームシーンへ遷移します。ゲームシーンは本ゲームのメインとなるシーンであり、プレイヤーは自機を操作して敵を倒しながらゲームを進めます。プレイヤーは初期状態でゲームを開始します。ゲーム中にプレイヤーのHPが0になるとゲームオーバーとなり、リザルトシーンへ遷移します。

- [ゲームシーン : GameScene_Specification.md](GameScene_Specification.md)
- [プレイヤー : Player_Specification.md](Player_Specification.md)

ゲームは「通常ステージ」と「ボスステージ」の2段階で進行します。

ゲーム開始後は通常ステージから始まります。通常ステージでは雑魚敵が出現します。出現した雑魚敵が画面外へ退場または撃破されるとボス出現の演出が始まり、ボスステージへ遷移します。一度ボスステージへ移行すると、通常ステージへ戻ることはありません。

ボスステージではボス敵が出現し、ボスのHPを0にするとゲームクリアとなります。ボス撃破演出後、ゲームクリア画面を表示します。ゲームクリアでもリザルトシーンへ遷移します。

#### リザルトシーン

ゲームオーバー、またはゲームクリアになるとリザルトシーンへ遷移します。リザルトシーンでは、ゲーム中の経過時間と獲得スコアを表示します。

また、本ゲームで唯一シーンを選択できる場面であり、「タイトルシーンへ戻る」または「ゲームシーンを最初から開始する」のいずれかを選択できます。

- [リザルトシーン : ResultScene_Specification](ResultScene_Specification.md)

## ゲームルール

### クリア条件
  - ボスを倒す
    - ゲームは通常ステージから開始され、雑魚敵を倒しながらステージを進めます
    - 通常ステージを突破するとボスステージへ遷移し、最終目標のボスが出現します
    - ボス敵の攻撃をいなしながら、プレイヤーが撃破されずに、ボスのHPを0にするとボスを撃破し、ゲームクリアとなります
### ゲームオーバー条件 
- プレイヤーのHPが0になる
   - 敵の弾に被弾し、HPが0になった場合ゲームオーバーになります。 
   - 基本的に弾の種類関係なく3回の被弾でHPが0になります。
- [プレイヤー : Player_Specification.md](Player_Specification.md)

### スコア
ゲームプレイの成果を示す指標です。
プレイヤーは以下の場合スコアを獲得します。減点されることはないです。
- 敵に弾を当てる
- 敵を撃破
- アイテムの取得

獲得したスコアは画面下部にリアルタイムで表示され、最終的な結果はリザルトシーンで確認できます
- [ゲームシーン : GameScene_Specification.md](GameScene_Specification.md)

### ゲーム経過時間
ゲーム開始から終了までの時間を計測します。ゲームプレイの効率性を示す指標です。
- タイトルシーンからゲームシーンへ遷移をゲーム開始、ゲームシーンからリザルトシーンへ遷移をゲーム終了としています。
- ゲーム経過時間は画面下部にリアルタイムで表示され、最終的な結果はリザルトシーンで確認できます
- [ゲームシーン : GameScene_Specification.md](GameScene_Specification.md)

### アイテム

- 敵を撃破するとアイテムをドロップします。
- 経験値と回復と無敵のアイテムがあります。
- アイテムに近づいて取得することでそれぞれ効果を発揮します。

### プレイヤー
 プレイヤーは青い自機を操作してゲームを進めていきます。攻撃と移動を操作することができます。
 - [プレイヤー : Player_Specification.md](Player_Specification.md)
#### 攻撃
- スペースキーを押し続けることで自機から弾の発射をすることができます。
- 発射にはパターンがあり、強化してくことができます。
- 強化されても発射操作は変わらず、スペースキーを押し続けることで発射できます。
#### ダメージ
- 敵の弾に被弾するとＨＰが1つ減ります。
- ＨＰが0になるとゲームオーバーとなります。
- 無敵アイテムを拾うことで一定時間ダメージを防ぐこともできます。
- 敵と直接衝突してもダメージは受けません。


#### 強化
- プレイヤーは撃破すると落とす経験値アイテムを使って弾の発射を強化していくことができます。
- 獲得した経験値(ショットパワー)によって、段階的に発射の強化があります。
- ゲーム進行中に獲得した経験値(ショットパワー)を失うことはありません。

### 敵
敵は3種類の雑魚敵と1種類のボス敵がいます。攻撃を行って倒すことができ、
ボス敵は倒すとゲームクリアになります。

#### 通常敵
- 基本的な敵キャラクターです。シンプルな掃射と水平移動を行ってくる敵です。
- 動きが読みやすく、避けて撃つゲームの基本を伝える役割があります。
- [通常敵 : NomalEnemy_Specification.md](NomalEnemy_Specification.md)
#### 追従敵
- プレイヤーを追ってくる移動と十字の4方向同時掃射を行ってくる敵です。
- 安置にとどまらせないようにプレイヤーへ圧をかける役割があります。
- [追従敵 : TraceEnemy_Specification.md](TraceEnemy_Specification.md)
#### チャージ敵
- 大きく円弧を描く放物線移動とチャージをしてプレイヤーを狙い撃ち攻撃してくる敵です。
- ボスステージでは、ボス敵の攻撃にあわせて強力な攻撃を行います。
- プレイヤー狙い撃ちで避けを意識させる役割があります。
- [チャージ敵 : ChargeEnemy_Specification.md](ChargeEnemy_Specification.md)
#### ボス敵
- ゲームの最終目標であり倒すとゲームクリアです。
- 雑魚敵よりも巨大でHPがとても多く、移動と攻撃がHP割合によって変化します。
- ボスのHPバーがボスステージでは表示されます。
- [ボス敵 : BossEnemy_Specification.md](BossEnemy_Specification.md)

#### 敵の出現
敵はゲームの進行に合わせて時間経過で出現します。ステージによって出現する敵が変わります
#### 通常ステージ
- 3種類の雑魚敵（通常・追従・チャージ）が出現します。
- 出現するパターンが3つあり、進行度に応じて変化します。
#### ボスステージ
- ボス敵が出現する専用ステージです。
- 基本的にボス敵と一騎打ちですが、ボス敵の攻撃（召喚）によって雑魚敵も出現します。

## システム操作(ゲームアプリの終了)

ESCAPEキーを押すと、どの条件下であってもゲームアプリケーションは強制終了します。

## 操作方法
各シーンで操作が違います。先述したゲームアプリ終了操作はどのシーンであっても有効です。

| キー | 動作 |
|------|------|

| キー | 動作 |

| キー | 動作 |
|------|--------|

## 設計の意図と背景
### コンセプト
- 過去に制作したシューティングゲームを、一から作り直す。
- 作り直す際に追加する要素はなし
- 設計・実装・テストまで含めた成果物を制作する
### 構造の概要
- 各クラス間の依存を最小限にした疎結合な設計を目指す
- 他クラスへの依存を減らし、役割を明確にする
- 保守性と可読性の向上を目的とする
