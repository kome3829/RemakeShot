# 概要
## ゲーム概要

プレイヤーを操作して敵を倒しながらステージを進み、
最後に出現するボスを撃破するとゲームクリアとなる
2D縦スクロールシューティングゲーム。

## 基本情報
| 項目 | 内容 |
|------|------|
| ゲームジャンル | 2Dシューティング |
| プレイ人数 | 1人 |
| 画面サイズ | 640×480px |
| 操作方法 | キーボード |

## ゲームのルール

### ゲームの流れ

本ゲームは、ゲーム開始からリザルトシーンまで一方向に進行するシーン構成となっています。基本的にプレイヤーが任意のシーンへ移動することはできず、ゲームの進行に合わせてシーンが遷移します。

#### タイトルシーン

ゲームを起動するとタイトルシーンが表示されます。タイトル画像と操作説明画像を確認した後、ゲームを開始します。タイトル画像から操作説明画像への遷移は一方向のみで、タイトル画像へ戻ることはできません。

- [タイトルシーン仕様書：Title_Scene_Specifications.md](Title_Scene_Specifications.md)

#### ゲームシーン

ゲームシーンでは、プレイヤーは初期状態でゲームを開始します。ゲーム中にプレイヤーのHPが0になるとゲームオーバーとなり、リザルトシーンへ遷移します。

- [ゲームシーン仕様書：GameSceneSpecifications.md](GameSceneSpecifications.md)
- [プレイヤー仕様書：Player_Specifications.md](Player_Specifications.md)

ゲームは「通常ステージ」と「ボスステージ」の2段階で進行します。

まず通常ステージが開始され、出現した雑魚敵がすべて退場（撃破または画面外への離脱）するとボスステージへ遷移します。一度ボスステージへ移行すると、通常ステージへ戻ることはありません。

ボスステージではボス敵が出現し、ボスのHPを0にするとゲームクリアとなります。ゲームクリアでもリザルトシーンへ遷移します。

#### リザルトシーン

ゲームオーバー、またはゲームクリアになるとリザルトシーンへ遷移します。リザルトシーンでは、ゲーム中の経過時間と獲得スコアを表示します。

また、本ゲームで唯一シーンを選択できる場面であり、「タイトルシーンへ戻る」または「ゲームシーンを最初から開始する」のいずれかを選択できます。

- [リザルトシーン仕様書：ResultScene_Specification.md](ResultScene_Specification.md)


### クリア条件
  - ボスを倒す
    - ボス敵はボスステージで出現します。ボス敵にはHPが3000あり、HPがゼロになると撃破となります。
### ゲームオーバー条件 
- プレイヤーのライフが0になる
   - 敵の弾に被弾し、HPが0になった場合ゲームオーバーになります。 
   - [プレイヤー仕様書：Player_Specifications.md](Player_Specifications.md)
## システム操作(ゲームアプリの終了)
ESCAPEキーを押すと、どの条件下であってもゲームアプリケーションは強制終了します。
## 操作方法
- 移動：矢印キー
- 攻撃：SPACEキー(ショット)
- アクション：シフトキー（低速移動）
## 設計の意図と背景
### コンセプト
- 過去に制作したシューティングゲームを、一から作り直す。
- 作り直す際に追加する要素はなし
- 設計・実装・テストまで含めた成果物を制作する
### 構造の概要
- 各クラス間の依存を最小限にした疎結合な設計を目指す
- 他クラスへの依存を減らし、役割を明確にする
- 保守性と可読性の向上を目的とする
## シーン構成
- タイトルシーン：ゲーム開始画面、操作説明画面
- ゲームシーン：ゲーム本編
- リザルトシーン：クリア・ゲームオーバー後の結果表示、シーン遷移の選択
