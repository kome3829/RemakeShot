# 概要
## ゲーム概要

プレイヤーを操作して敵を倒しながらステージを進み、
最後に出現するボスを撃破するとゲームクリアとなる
2D縦スクロールシューティングゲーム。

## 基本情報
| 項目 | 内容 |
|------|------|
| ゲームジャンル | 2Dシューティング |
| プレイ人数 | 1人 |
| 画面サイズ | 640×480px |
| 操作方法 | キーボード |
---

## ゲームのルール（クリアとゲームオーバーの条件）
### クリア条件
  - ボスを倒す
### ゲームオーバー条件 
- プレイヤーのライフが0になる 
## 操作方法
- 移動：矢印キー
- 攻撃：SPACEキー(ショット)
- アクション：シフトキー（低速移動）
- システム操作：ESCAPEキー（ゲーム終了）
## 設計の意図と背景
### コンセプト
- 過去に制作したシューティングゲームを、一から作り直す。
- 作り直す際に追加する要素はなし
- 設計・実装・テストまで含めた成果物を制作する
### 構造の概要
- 各クラス間の依存を最小限にした疎結合な設計を目指す
- 他クラスへの依存を減らし、役割を明確にする
- 保守性と可読性の向上を目的とする
## シーン構成
- タイトルシーン：ゲーム開始画面
- ゲームシーン：ゲーム本編
- リザルトシーン：クリア・ゲームオーバー後の結果表示
