# リザルトシーン

## シーン概要
ゲームプレイの結果を表示するシーン  
ゲームプレイの結果を表示し、
タイトルシーンまたはゲームシーンへの遷移を選択できる。
---
## シーンフロー

```text
ゲームシーン
    │
    ▼
リザルトシーン
    │
    ├─ Rキー押下
    │      ▼
    │  ゲームシーン
    │
    ├─ Tキー押下
    │      ▼
    │  タイトルシーン
    │
    └─ Escapeキー押下
           ▼
       ゲーム終了
```

## 表示内容
| 表示要素 | サイズ | 
|----------|----------|
| リザルト背景画像 | 画面全体　640×480px |

| タイトルテキスト（SCORE） | フォントサイズ　(RESULT_TITLE_FONT_SIZE):45  |
| 最終ゲームスコア | フォントサイズ　(RESULT_SCORE_FONT_SIZE):55 |

| ゲーム内経過プレイ時間 | フォントサイズ　(RESULT_SCORE_FONT_SIZE):55 |
| 操作説明テキスト（Retry：Rキー） | フォントサイズ　(RESULT_DEFAULT_FONT_SIZE):16 |
| 操作説明テキスト（Title：Tキー） | フォントサイズ　(RESULT_DEFAULT_FONT_SIZE):16 | 
---
- リザルト背景画像
  - 1つ画面全体に背景画像を表示する
- リザルト表示ボックス
  - ゲーム結果（スコア・タイム）を表示する

## 演出

### スコア・時間の表示仕様
- スコア　カウントアップしていく
  -  0から最終スコアまで3フレーム毎に加算表示する
  - 最終スコアまで加算するとカウントアップ終了
  - 目標の桁が0でもカウントアップさせる。動きに統一感をもたせるため
- タイム　点滅表示する
  - 30フレーム目から表示を開始し、60フレーム目でリセットする
  - スコアのカウントアップ開始と同時にタイムの点滅表示を開始する
## 操作

### 基本操作
| キー | 動作 |
|------|------|
| R | ゲームシーンへ遷移 |
| T | タイトルシーンへ遷移 |

### 特殊操作
| キー | 動作 |
|------|------|
| Escape | ゲーム終了 |

---

## 開始時処理

- データクラスからゲームシーンで保存されたプレイ結果を取得する
  - スコア
  - タイム
- 表示用カウンタ初期化
---

## 更新処理

- 入力操作判定
- スコア表示のカウントアップ更新
- タイム表示の点滅更新
- 表示更新
---

## 終了条件

- Rキーが押されたとき
- Tキーが押されたとき
- Escapeキーが押されたとき
---

## 遷移先

| 条件 | 遷移先 |
|------|--------|
| Rキー押下 | ゲームシーン |
| Tキー押下 | タイトルシーン |
| ESCAPEキー押下 | ゲーム終了 |

## 使用データ

| データ | 内容 |
|--------|------|
| Score | 最終スコア |
| Time | プレイ時間 |
